# Структура меню и навигации

## Дата: 12 марта 2026 г.

## Концепция

Разделение интерфейса на публичную часть и личные инструменты пользователя.

---

## Структура меню

### Публичная часть (доступно всем)
```
📊 Калькулятор
📦 Стойки
... (другие публичные разделы)
```

### Личные инструменты (видно только авторизованному пользователю)
```
🔧 Мои инструменты
   ├─ Быстрый расчёт
   ├─ Мои заметки
   └─ ... (расширяемо)
```

**Расположение:** иконка профиля/аватарки в хедере → выпадающее меню:
- 👤 Профиль
- 🔧 Мои инструменты
- ⚙️ Настройки приложения (админка)
- 🚪 Выход

---

## Принципы

1. **Админка** — управление приложением (пользователи, логи, настройки)
2. **Мои инструменты** — личные рабочие инструменты пользователя (замена внешним табличкам)
3. **Публичная часть** — основной функционал для всех пользователей
