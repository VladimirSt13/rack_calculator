# Аккумуляторы: Бренды и Модели

## Дата: 12 марта 2026 г.

## Идея

Добавить возможность выбора аккумулятора из каталога с автозаполнением параметров.

---

## Функционал

### Для пользователя (калькулятор)

1. **Выбор бренда** → выпадающий список
2. **Выбор модели** → каскадно после выбора бренда
3. **Автозаполнение** при выборе модели:
   - Размеры (Д×Ш×В)
   - Вес
   - ГЕп (ёмкость)
4. **Ручной ввод остаётся** — можно перебить автозаполнение или ввести свои данные

### Для админа (управление каталогом)

- Добавление/редактирование брендов
- Добавление/редактирование моделей (привязано к бренду)
- Удаление (если не используются)

---

## Структура БД

```sql
battery_brands
├── id
├── name (Leocham, Delta, Challenger, etc.)
└── created_at

battery_models
├── id
├── brand_id (FK → battery_brands.id)
├── model_name
├── length
├── width
├── height
├── weight
├── capacity (ГЕп)
├── created_at
└── updated_at
```

## Примечания

- Ручной ввод размеров должен оставаться доступным всегда
- Автозаполнение не блокирует поля — пользователь может перебить значения
- При сбросе формы — очищать и выбор бренда/модели тоже
