# 📤 Експорт комплектів стелажів в Excel

**Дата додавання:** 7 березня 2026  
**Статус:** ✅ Реалізовано

---

## 📋 Опис

Функція експорту комплектів стелажів в Excel форматі з можливістю включення/виключення цін.

### Особливості

- ✅ **Один аркуш** - вся інформація на одній сторінці
- ✅ **Опція цін** - чекбокс "Додати ціни в експорт"
- ✅ **Експорт нового комплекту** - без збереження в БД
- ✅ **Експорт збереженого комплекту** - з БД по ID
- ✅ **Комплектація стелажів** - детальна інформація по компонентах
- ✅ **Український формат чисел** - кома як розділовий знак (наприклад: `1500,00`)

## 🔧 Технічна реалізація

### Server

#### Endpoint: Експорт збереженого комплекту

```
GET /api/rack-sets/:id/export
```

**Query parameters:**
- `includePrices` (boolean) - чи включати ціни

**Response:**
- Content-Type: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
- Файл Excel з ім'ям: `{назва}_з_цінами.xlsx` або `{назва}.xlsx`

**Структура Excel:**


#### Endpoint: Експорт нового комплекту

```
POST /api/rack-sets/export
```

**Body:**
```json
{
  "racks": [...],
  "includePrices": false
}
```

**Response:**
- Файл Excel з ім'ям: `новий_комплект_з_цінами.xlsx` або `новий_комплект.xlsx`

#### API: rackSetsApi

**Методи:**

```typescript
// Експорт збереженого комплекту
export: async (id: number, includePrices: boolean = false) => {
  const response = await api.get(`/rack-sets/${id}/export`, {
    params: { includePrices },
    responseType: 'arraybuffer',
  });
  return response.data;
}

// Експорт нового комплекту
exportNew: async (racks: RackSetItem[], includePrices: boolean = false) => {
  const response = await api.post('/rack-sets/export', { racks, includePrices }, {
    responseType: 'arraybuffer',
  });
  return response.data;
}
```

## 🎯 Сценарії використання

### Сценарій 1: Експорт нового комплекту

1. Користувач розраховує стелажі
2. Натискає "Зберегти комплект"
3. Відкривається модалка з формою
4. Користувач заповнює назву та об'єкт
5. **Вибирає чекбокс "Додати ціни в експорт"** (опціонально)
6. Натискає **"Експорт в Excel"**
7. Файл завантажується

### Сценарій 2: Експорт збереженого комплекту

1. Користувач переходить в "Адмін-панель" → "Комплекти стелажів"
2. Знаходить потрібний комплект
3. Натискає кнопку "Експорт" (потрібно додати в UI)
4. Обирає опцію цін
5. Файл завантажується

## 📝 Примітки

1. **Формат цін:**
   - "Ціна без ізоляторів" - з масиву `prices` з type='без_ізоляторів'
   - "Нульова ціна" - з масиву `prices` з type='нульова'
   - "Загальна нульова ціна" = Нульова ціна × Кількість

2. **Іменування файлів:**
   - Без цін: `{назва}.xlsx`
   - З цінами: `{назва}_з_цінами.xlsx`

3. **Безпека:**
   - Endpoint вимагає автентифікації
   - Користувач може експортувати тільки свої комплекти
